# Gregory Tweaks For Crafting

`Gregory Tweaks For Crafting` is a Minecraft 1.7.10 mod that adds quick crafting-grid utilities to supported crafting GUIs.

## Features

- Rotate the crafting grid.
- Balance items across the grid.
- Clear the crafting grid.
- Compress / decompress shortcuts (when supported).
- Optional on-screen tweak buttons and keybinds.

## Provenance

This mod was derived from the MIT-licensed release of Crafting Tweaks by BlayTheNinth.

The mod name was changed to `Gregory Tweaks For Crafting` at the request of the original author.
